---- HOW SET UP THE APPLICATION AND RUN IT ON A LOCAL ENVIRONMENT ----

  -- PREREQUISITES --
    - Installation of Docker
    - Installation of Visual Studio in any of its versions (As for this app development Visual Studio 2026 Insiders Community Edition)
    
  -- LOCAL ENVIRONMENT SETUP --
    1. Run the .yml file containing the scripts to generate and initialise the SQL Container in Docker and the database that will be conected to the app.
      'docker compose up -d' to start the service and 'docker compose down' to stop the service.
    2. Open the project in Visual Studio and run the app from there.
    3. Play around with it!
